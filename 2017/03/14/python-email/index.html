论如何使用python发一封写着自己IP地址的邮件

近来收到动态IP困扰，远程桌面使用的略不顺手，总是因为奇奇怪怪的ip改换导致连接失败。所以写个脚本用于读取本地IP并发送给自己的邮箱，方便你我他嗯。

思路

  1. 读取IP
  2. 生成邮件
  3. 发送邮件

实践

读取IP

这一步倒是简单的很，windows的命令行有这样一条命令：ipconfig

这条命令是用于读取本地的实时IP，可以读取所有网卡（包括虚拟网卡的地址），因此不需要我再找什么模块或者插件了。
